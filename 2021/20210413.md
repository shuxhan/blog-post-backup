title: 交换两个元素的位置
date: 2021-04-13

---
使用 jquery 交换两个元素的位置，重点是 `insertBefore` 属性。

```html
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Document</title>
<script src="https://cdn.shuxhan.com/jquery3.6.0.js"></script>
</head>
<body>
  <ul>
    <li>1</li>
    <li>2</li>
  </ul>
<script>
  $(function(){
    var $one_li = $("ul li:nth-child(1)");
    var $two_li = $("ul li:nth-child(2)");
    $two_li.insertBefore($one_li);
  });
</script>
</body>
</html>
```
